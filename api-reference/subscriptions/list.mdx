---
title: "List Subscriptions"
openapi: "GET /subscriptions"
---

List subscriptions with optional filters for status and customer.

<CodeGroup>

```bash cURL
curl "https://api.billing.io/v1/subscriptions?status=active&limit=10" \
  -H "Authorization: Bearer $BILLING_API_KEY"
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/subscriptions?status=active&limit=10",
  { headers: { "Authorization": `Bearer ${process.env.BILLING_API_KEY}` } }
);
const { data, has_more, next_cursor } = await response.json();
```
