---
title: "List Subscription Plans"
openapi: "GET /subscriptions/plans"
---

List all subscription plans. Optionally filter by status.

<CodeGroup>

```bash cURL
curl "https://api.billing.io/v1/subscriptions/plans?status=active" \
  -H "Authorization: Bearer $BILLING_API_KEY"
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/subscriptions/plans?status=active",
  { headers: { "Authorization": `Bearer ${process.env.BILLING_API_KEY}` } }
);
const { data } = await response.json();
```

</CodeGroup>
