---
title: "Create an Entitlement"
openapi: "POST /subscriptions/entitlements"
---

Attach a feature entitlement to a subscription plan.

<CodeGroup>

```bash cURL
curl -X POST https://api.billing.io/v1/subscriptions/entitlements \
  -H "Authorization: Bearer $BILLING_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "plan_id": "plan_a1b2c3d4e5f6",
    "feature_key": "api_requests",
    "value_type": "numeric",
    "value_numeric": 10000
  }'
```

```javascript Node.js
const response = await fetch("https://api.billing.io/v1/subscriptions/entitlements", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.BILLING_API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    plan_id: "plan_a1b2c3d4e5f6",
    feature_key: "api_requests",
    value_type: "numeric",
    value_numeric: 10000,
  }),
});
const entitlement = await response.json();
```
