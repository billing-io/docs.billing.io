---
title: "Check an Entitlement"
openapi: "GET /subscriptions/entitlements/check"
---

Check if a customer has access to a specific feature based on their active subscription.

<CodeGroup>

```bash cURL
curl "https://api.billing.io/v1/subscriptions/entitlements/check?customer_id=cus_a1b2c3d4e5f6&feature_key=api_requests" \
  -H "Authorization: Bearer $BILLING_API_KEY"
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/subscriptions/entitlements/check?customer_id=cus_a1b2c3d4e5f6&feature_key=api_requests",
  { headers: { "Authorization": `Bearer ${process.env.BILLING_API_KEY}` } }
);
const result = await response.json();
console.log(result.has_access); // true
console.log(result.value_numeric); // 10000
```

</CodeGroup>

```json Response
{
  "has_access": true,
```
