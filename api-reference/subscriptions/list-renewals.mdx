---
title: "List Renewals"
openapi: "GET /subscriptions/renewals"
---

List subscription renewals. Filter by status or subscription ID.

<CodeGroup>

```bash cURL
curl "https://api.billing.io/v1/subscriptions/renewals?subscription_id=sub_a1b2c3d4e5f6&status=paid" \
  -H "Authorization: Bearer $BILLING_API_KEY"
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/subscriptions/renewals?subscription_id=sub_a1b2c3d4e5f6",
  { headers: { "Authorization": `Bearer ${process.env.BILLING_API_KEY}` } }
);
const { data } = await response.json();
```
