---
title: "Create a Subscription"
openapi: "POST /subscriptions"
---

Subscribe a customer to a plan. Creates the first renewal automatically.

<CodeGroup>

```bash cURL
curl -X POST https://api.billing.io/v1/subscriptions \
  -H "Authorization: Bearer $BILLING_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "customer_id": "cus_a1b2c3d4e5f6",
    "plan_id": "plan_a1b2c3d4e5f6",
    "payment_method_id": "pm_a1b2c3d4e5f6"
  }'
```

```javascript Node.js
const response = await fetch("https://api.billing.io/v1/subscriptions", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.BILLING_API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    customer_id: "cus_a1b2c3d4e5f6",
    plan_id: "plan_a1b2c3d4e5f6",
    payment_method_id: "pm_a1b2c3d4e5f6",
  }),
});
const subscription = await response.json();
console.log(subscription.id); // "sub_a1b2c3..."
```
