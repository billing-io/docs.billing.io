---
title: "Update a Subscription"
openapi: "PATCH /subscriptions/{id}"
---

Cancel, pause, or resume a subscription.

<CodeGroup>

```bash cURL
curl -X PATCH https://api.billing.io/v1/subscriptions/sub_a1b2c3d4e5f6 \
  -H "Authorization: Bearer $BILLING_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{ "action": "cancel" }'
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/subscriptions/sub_a1b2c3d4e5f6",
  {
    method: "PATCH",
    headers: {
      "Authorization": `Bearer ${process.env.BILLING_API_KEY}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({ action: "cancel" }),
  }
);
const subscription = await response.json();
console.log(subscription.cancel_at_period_end); // true
```

</CodeGroup>
