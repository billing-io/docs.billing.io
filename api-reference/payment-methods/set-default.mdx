---
title: "Set as Default Payment Method"
openapi: "POST /payment-methods/{id}/default"
---

Set a payment method as the default for your organization.

<CodeGroup>

```bash cURL
curl -X POST https://api.billing.io/v1/payment-methods/pm_a1b2c3d4e5f6/default \
  -H "Authorization: Bearer $BILLING_API_KEY"
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/payment-methods/pm_a1b2c3d4e5f6/default",
  {
    method: "POST",
    headers: { "Authorization": `Bearer ${process.env.BILLING_API_KEY}` },
  }
);
const pm = await response.json();
console.log(pm.is_default); // true
```

</CodeGroup>
