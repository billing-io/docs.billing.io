---
title: "Update a Payment Method"
openapi: "PATCH /payment-methods/{id}"
---

Update a payment method's display name, confirmation requirements, or amount limits.

<CodeGroup>

```bash cURL
curl -X PATCH https://api.billing.io/v1/payment-methods/pm_a1b2c3d4e5f6 \
  -H "Authorization: Bearer $BILLING_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "display_name": "Updated USDT Tron",
    "min_amount_usd": 10.00,
    "max_amount_usd": 5000.00
  }'
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/payment-methods/pm_a1b2c3d4e5f6",
  {
    method: "PATCH",
    headers: {
      "Authorization": `Bearer ${process.env.BILLING_API_KEY}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      display_name: "Updated USDT Tron",
      min_amount_usd: 10.00,
      max_amount_usd: 5000.00,
    }),
  }
);
const pm = await response.json();
```
