---
title: "Create a Payment Method"
openapi: "POST /payment-methods"
---

Register a new payment method (chain + token + wallet). Checkouts will use this to generate deposit addresses.

<CodeGroup>

```bash cURL
curl -X POST https://api.billing.io/v1/payment-methods \
  -H "Authorization: Bearer $BILLING_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "wallet_id": "wal_abc123",
    "chain": "tron",
    "token": "USDT",
    "display_name": "Primary USDT on Tron",
    "is_default": true
  }'
```

```javascript Node.js
const response = await fetch("https://api.billing.io/v1/payment-methods", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.BILLING_API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    wallet_id: "wal_abc123",
    chain: "tron",
    token: "USDT",
    display_name: "Primary USDT on Tron",
    is_default: true,
  }),
});
const pm = await response.json();
console.log(pm.id); // "pm_a1b2c3..."
```
