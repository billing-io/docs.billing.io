---
title: "Create a Checkout"
openapi: "POST /checkouts"
---

Create a new checkout session. Returns a unique deposit address where the customer sends crypto.

<CodeGroup>

```bash cURL
curl -X POST https://api.billing.io/v1/checkouts \
  -H "Authorization: Bearer $BILLING_API_KEY" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: $(uuidgen)" \
  -d '{
    "amount_usd": 49.99,
    "chain": "tron",
    "token": "USDT",
    "metadata": {
      "order_id": "ord_12345"
    }
  }'
```

```javascript Node.js
const checkout = await fetch("https://api.billing.io/v1/checkouts", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.BILLING_API_KEY}`,
    "Content-Type": "application/json",
    "Idempotency-Key": crypto.randomUUID(),
  },
  body: JSON.stringify({
    amount_usd: 49.99,
    chain: "tron",
    token: "USDT",
    metadata: { order_id: "ord_12345" },
  }),
});
const data = await checkout.json();
console.log(data.deposit_address);
```
