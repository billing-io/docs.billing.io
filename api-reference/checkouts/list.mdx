---
title: "List Checkouts"
openapi: "GET /checkouts"
---

Returns a paginated list of checkouts, newest first. Supports filtering by status.

<CodeGroup>

```bash cURL
curl "https://api.billing.io/v1/checkouts?limit=10&status=confirmed" \
  -H "Authorization: Bearer $BILLING_API_KEY"
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/checkouts?limit=10&status=confirmed",
  { headers: { "Authorization": `Bearer ${process.env.BILLING_API_KEY}` } }
);
const { data, has_more, next_cursor } = await response.json();
```
