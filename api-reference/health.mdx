---
title: "Health Check"
openapi: "GET /health"
---

Check if the billing.io API is operational.

<CodeGroup>

```bash cURL
curl https://api.billing.io/v1/health
```

```javascript Node.js
const response = await fetch("https://api.billing.io/v1/health");
const data = await response.json();
console.log(data.status); // "healthy"
```

</CodeGroup>

```json Response
{
  "status": "healthy",
  "version": "1.0.0"
}
```
