---
title: "Delete a Webhook Endpoint"
openapi: "DELETE /webhooks/{webhook_id}"
---

Delete a webhook endpoint. It will stop receiving events immediately.

<CodeGroup>

```bash cURL
curl -X DELETE https://api.billing.io/v1/webhooks/we_a1b2c3d4e5f6a7b8c9d0e1f2 \
  -H "Authorization: Bearer $BILLING_API_KEY"
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/webhooks/we_a1b2c3d4e5f6a7b8c9d0e1f2",
  {
    method: "DELETE",
    headers: { "Authorization": `Bearer ${process.env.BILLING_API_KEY}` },
  }
);
// Returns 204 No Content on success
```

</CodeGroup>
