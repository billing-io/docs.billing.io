---
title: "List Webhook Endpoints"
openapi: "GET /webhooks"
---

List all registered webhook endpoints.

<CodeGroup>

```bash cURL
curl "https://api.billing.io/v1/webhooks?limit=10" \
  -H "Authorization: Bearer $BILLING_API_KEY"
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/webhooks?limit=10",
  { headers: { "Authorization": `Bearer ${process.env.BILLING_API_KEY}` } }
);
const { data, has_more } = await response.json();
```

</CodeGroup>
