---
title: "Get a Webhook Endpoint"
openapi: "GET /webhooks/{webhook_id}"
---

Retrieve details for a specific webhook endpoint.

<CodeGroup>

```bash cURL
curl https://api.billing.io/v1/webhooks/we_a1b2c3d4e5f6a7b8c9d0e1f2 \
  -H "Authorization: Bearer $BILLING_API_KEY"
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/webhooks/we_a1b2c3d4e5f6a7b8c9d0e1f2",
  { headers: { "Authorization": `Bearer ${process.env.BILLING_API_KEY}` } }
);
const webhook = await response.json();
console.log(webhook.url);
console.log(webhook.events);
```

</CodeGroup>
