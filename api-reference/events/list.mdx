---
title: "List Events"
openapi: "GET /events"
---

List webhook events. Useful for replaying missed webhooks or debugging. Filter by event type or checkout ID.

<CodeGroup>

```bash cURL
curl "https://api.billing.io/v1/events?type=checkout.completed&limit=10" \
  -H "Authorization: Bearer $BILLING_API_KEY"
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/events?type=checkout.completed&limit=10",
  { headers: { "Authorization": `Bearer ${process.env.BILLING_API_KEY}` } }
);
const { data, has_more, next_cursor } = await response.json();
```
