---
title: "List Payout Intents"
openapi: "GET /payouts"
---

List payout intents. Filter by status or due date range.

<CodeGroup>

```bash cURL
curl "https://api.billing.io/v1/payouts?status=approved&limit=10" \
  -H "Authorization: Bearer $BILLING_API_KEY"
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/payouts?status=approved&limit=10",
  { headers: { "Authorization": `Bearer ${process.env.BILLING_API_KEY}` } }
);
const { data, has_more, next_cursor } = await response.json();
```
