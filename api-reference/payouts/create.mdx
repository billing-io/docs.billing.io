---
title: "Create a Payout Intent"
openapi: "POST /payouts"
---

Create a new payout intent in draft status.

<CodeGroup>

```bash cURL
curl -X POST https://api.billing.io/v1/payouts \
  -H "Authorization: Bearer $BILLING_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "recipient_address": "TXyz123abc456def789ghi",
    "chain": "tron",
    "token": "USDT",
    "amount": 500.00,
    "currency": "USD",
    "description": "Vendor payment - January",
    "due_date": "2025-02-01T00:00:00Z"
  }'
```

```javascript Node.js
const response = await fetch("https://api.billing.io/v1/payouts", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.BILLING_API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    recipient_address: "TXyz123abc456def789ghi",
    chain: "tron",
    token: "USDT",
    amount: 500.00,
    currency: "USD",
    description: "Vendor payment - January",
    due_date: "2025-02-01T00:00:00Z",
  }),
});
const payout = await response.json();
console.log(payout.id); // "po_a1b2c3..."
console.log(payout.status); // "draft"
```
