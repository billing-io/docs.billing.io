---
title: "Create a Payment Link"
openapi: "POST /payment-links"
---

Create a shareable payment link that generates a checkout when visited.

<CodeGroup>

```bash cURL
curl -X POST https://api.billing.io/v1/payment-links \
  -H "Authorization: Bearer $BILLING_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "amount_usd": 29.99,
    "payment_method_id": "pm_a1b2c3d4e5f6",
    "description": "Pro Plan - Monthly"
  }'
```

```javascript Node.js
const response = await fetch("https://api.billing.io/v1/payment-links", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.BILLING_API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    amount_usd: 29.99,
    payment_method_id: "pm_a1b2c3d4e5f6",
    description: "Pro Plan - Monthly",
  }),
});
const link = await response.json();
console.log(link.url); // "https://pay.billing.io/pl_..."
```
