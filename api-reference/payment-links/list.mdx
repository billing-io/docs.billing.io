---
title: "List Payment Links"
openapi: "GET /payment-links"
---

List all payment links for your organization.

<CodeGroup>

```bash cURL
curl "https://api.billing.io/v1/payment-links?limit=10" \
  -H "Authorization: Bearer $BILLING_API_KEY"
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/payment-links?limit=10",
  { headers: { "Authorization": `Bearer ${process.env.BILLING_API_KEY}` } }
);
const { data, has_more } = await response.json();
```

</CodeGroup>
