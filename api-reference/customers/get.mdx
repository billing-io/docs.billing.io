---
title: "Get a Customer"
openapi: "GET /customers/{customer_id}"
---

Retrieve a single customer by ID.

<CodeGroup>

```bash cURL
curl https://api.billing.io/v1/customers/cus_a1b2c3d4e5f6a7b8c9d0e1f2 \
  -H "Authorization: Bearer $BILLING_API_KEY"
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/customers/cus_a1b2c3d4e5f6a7b8c9d0e1f2",
  { headers: { "Authorization": `Bearer ${process.env.BILLING_API_KEY}` } }
);
const customer = await response.json();
console.log(customer.email); // "alice@example.com"
```

</CodeGroup>
