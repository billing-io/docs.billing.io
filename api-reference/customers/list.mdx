---
title: "List Customers"
openapi: "GET /customers"
---

Returns a paginated list of customers. Supports searching by email, name, or external_id.

<CodeGroup>

```bash cURL
curl "https://api.billing.io/v1/customers?limit=10&search=alice" \
  -H "Authorization: Bearer $BILLING_API_KEY"
```

```javascript Node.js
const response = await fetch(
  "https://api.billing.io/v1/customers?limit=10&search=alice",
  { headers: { "Authorization": `Bearer ${process.env.BILLING_API_KEY}` } }
);
const { data, has_more } = await response.json();
```
